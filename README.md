# GameLog

A client-side web app to track your video game library built with Svelte.

## Features

- Track games across statuses: Playing, Completed, Dropped
- Search games using the RAWG API (cover images, genres, platforms)
- Rate games with 1–5 stars and write personal reviews
- Filter by status, genre and platform
- Mark games as favorites
- Light / dark theme
- All data persisted in localStorage

## Tech Stack

- [Svelte 5](https://svelte.dev/) + [Vite](https://vitejs.dev/)
- [RAWG Video Games Database API](https://rawg.io/apidocs)

## Run locally

```bash
npm install
npm run dev
```
